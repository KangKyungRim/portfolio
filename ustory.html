UStory

사용자가 친구들과의 추억을 기록할 수 있는 다이어리형 서비스입니다.
UStory 프로젝트를 통해 프론트엔드 배포 방법을 배우고, Redux-Toolkit을 사용하여 상태 관리의 중요성을 깨닫게 되었습니다.

기간2024.05~2024.06 (4주)

상세 역할 - 서비스 디자인 부터 프론트엔드 기능 구현까지 업무 수행
- 사용자 관련, 친구 관련, 소셜 로그인 파트 담당
- 소셜 로그인 구현으로 로그인 성공률 100% 향상

개발 환경 및 언어JavaScript, React, React Router, Vite, PWA, Node,js, Axios, AWS S3, Spring boot, Redis, MySQL
