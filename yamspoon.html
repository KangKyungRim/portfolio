YAM Spoon

내 냉장고 속 재료들로 만들 수 있는 레시피들을 제공하는 서비스입니다.
YAM Spoon 프로젝트를 통해 git 관리의 중요성을 배우고, React의 편리함에 대해 느낄뿐 아니라 관련된 라이브러리 등
개발하는데 편리한 프로그램들을 배우게 되었습니다.

기간2024.04~2024.04 (3주)

상세 역할 - 서비스 디자인 부터 프론트엔드 기능 구현까지 업무 수행
- 마이 페이지, 레시피 상세, 이외에 공통 컴포넌트 작업
- 컴포넌트 재사용성 향상으로 개발 생산성 증가

개발 환경 및 언어 JavaScript, React, React Router, CRA, Node,js, Axios, EXPRESS, MONGODB, AMAZON S3, PM2
